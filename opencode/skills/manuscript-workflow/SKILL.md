# Skill: manuscript-workflow

## Workflow: Three Passes

### 1. Outline Pass
- Create or refine topic-sentence outline per section
- Confirm with user before drafting
- Use `<!-- TODO: ... -->` for missing pieces

### 2. Draft Pass
- Fill paragraphs following hamburger structure
- Prioritize idea flow over polish
- Use `<!-- TODO: cite -->` for missing references

### 3. Revision Pass
- Tighten language and improve flow
- Check statistical consistency and terminology
- Ensure results prose aligns with analysis code

## Collaboration Rules

- Confirm scope first (target journal, word limit, deadline)
- Before large rewrites, provide:
  - Topic-sentence outline (new work)
  - Reverse outline (existing drafts)
- Defer to user on substantive choices:
  - Which limitations to emphasize
  - Strength of claims
  - What belongs in main text vs supplements
