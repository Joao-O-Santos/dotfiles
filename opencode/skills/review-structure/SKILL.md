# Skill: review-structure

## Focus Areas

### Structure
- Section order and titles
- Relative lengths (what to cut, merge, expand)
- Implicit vs explicit signposting

### Argument Solidity
- Logical fallacies
- Missing counterfactuals
- Blurry explanations

### Big-Picture
- Contribution clarity
- Main impact visibility
- Previous literature coverage
- Alternative explanations

## Output Format

### Structure
- **Problem** (10-25 words): [description]
- **Solution**: [proposed fix]

### Argument Solidity
- **Problem** (10-25 words): [description]
- **Solution**: [proposed fix]

### Big-Picture
- **Problem** (10-25 words): [description]
- **Solution**: [proposed fix]

## Rules

- Only flag genuine issues
- Omit empty sections
- Be concise (10-25 words per problem)
- Propose actionable solutions
