# Skill: review-structure

## Focus Areas

### Structure
- Section order and titles
- Relative lengths (what to cut, merge, expand)
- Implicit vs explicit signposting

### Argument Solidity
- Logical fallacies
- Missing counterfactuals
- Blurry explanations

### Big-Picture
- Contribution clarity
- Main impact visibility
- Previous literature coverage
- Alternative explanations
